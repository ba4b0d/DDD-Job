# بازطراحی Spaghetti — ادمین مینیمال / فیوچریستیک

**فایل Penpot:** صفحه `۰۳ ادمین · مینیمال`  
**تاریخ:** 2026-07-19  
**جهت:** فارسی · RTL · فقط برند `Spaghetti` لاتین  
**پس‌زمینه:** نقطه‌ای (dot texture) + هاله اکسنت بالا

## زبان طراحی

| مفهوم | مقدار |
|--------|--------|
| پس‌زمینه | `#0B0B12` |
| سطح / سایدبار | `#12121A` |
| کارت شیشه‌ای | `#181822` + حاشیه `#2A2A38` + سایه نرم |
| پنل | `#1C1C28` |
| متن | `#F4F4F5` · muted `#8B8B9A` · dim `#5C5C6E` |
| اکسنت | `#818CF8` (glow) · ثانویه cyan `#22D3EE` |
| موفقیت / هشدار / خطر | `#34D399` / `#FBBF24` / `#F87171` |
| شعاع | ۱۰–۱۶px · دکمه‌ها ۱۰ · کارت‌ها ۱۶ |
| RTL | `row-reverse` · متن `direction=rtl` · سایدبار راست |

## صفحات (همه با یک شل ادمین)

1. **داشبورد** — KPI + محصولات اخیر + اقدام سریع  
2. **محصولات** — جدول + جستجو + «+ محصول جدید»  
3. **افزودن محصول** — فرم شیشه‌ای (پایه / مواد / زمان) + پیش‌نمایش قیمت زنده  
4. **مواد** — جدول قیمت/کیلو و ضایعات  
5. **افزودن ماده** — نام، نوع، قیمت/کیلو، ضایعات، رنگ  
6. **ماشین‌ها** — وات، عمر، نگهداری  
7. **افزودن ماشین** — فرم مشخصات  
8. **ماشین حساب** — پارامترها + قیمت پیشنهادی  
9. **تنظیمات** — markup، بالاسری، برق، لوگو  
10. **دسته‌بندی‌ها** — لیست + افزودن  

شل مشترک: سایدبار راست (برند + ناو + وضعیت فعال) · نوار بالا · پس‌زمینه بافت‌دار.

## پیاده‌سازی در React (پیشنهاد)

```css
/* dark admin tokens */
--bg: #0B0B12;
--surface: #12121A;
--elevated: #181822;
--panel: #1C1C28;
--line: #2A2A38;
--text: #F4F4F5;
--muted: #8B8B9A;
--accent: #818CF8;
--accent-2: #22D3EE;
/* texture */
body.admin::before {
  content: '';
  position: fixed; inset: 0; pointer-events: none;
  background-image: radial-gradient(rgba(255,255,255,0.04) 1px, transparent 1px);
  background-size: 32px 32px;
  opacity: 1;
}
/* top glow */
body.admin::after {
  content: '';
  position: fixed; top: 0; left: 0; right: 0; height: 180px;
  background: radial-gradient(ellipse at 20% 0%, rgba(129,140,248,0.12), transparent 60%),
              radial-gradient(ellipse at 90% 0%, rgba(34,211,238,0.06), transparent 50%);
  pointer-events: none;
}
```

- همه صفحات ادمین از `Layout` + `Sidebar` یکسان  
- فرم افزودن: گرید ۲ ستونه کارت‌های شیشه‌ای  
- دکمه primary با `box-shadow: 0 4px 18px rgba(129,140,248,0.4)`  
- جداول: ردیف‌های ۴۸–۵۲px، badge وضعیت
